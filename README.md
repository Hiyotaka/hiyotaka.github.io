# Haoyu Zhai Homepage

Personal homepage built with React + Vite.

## Local Development

1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`
3. Build production files:
   `npm run build`

## Deploy

GitHub Pages deployment is automated via:

- `.github/workflows/deploy-pages.yml`

Push to `main` and GitHub Actions will build and deploy `dist/` automatically.
